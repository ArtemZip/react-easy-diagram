# react-easy-diagram

## Install

```bash
npm install --save react-easy-diagram
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-easy-diagram'
import 'react-easy-diagram/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
